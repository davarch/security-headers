# Security Headers for Laravel

## Installation
```bash
composer require davarch/security-headers
```

## Usage
```bash
php artisan vendor:publish --provider="Davarch\SecurityHeaders\Providers\PackageServiceProvider" --tag="config"
```

## Code style
```bash 
composer pint
```

## Analyse
```bash 
composer stan
```

## Tests
```bash
composer test
```

```bash
composer types
```

```bash
composer coverage
```
